The page creates a personalized ranking based on yor performance, comparing to other dancers you have competed with. Your scores are always 0, if someone performs better than you at a given event, gets positive score, if worst negative score. The scores are weighted with the time passed since that competition. The page also maintains an accuracy score to each dancer. Accuracy score is lower if you haven't competed with the given dancer since a while. Dancers with the lowest accuracy score are filtered out.
